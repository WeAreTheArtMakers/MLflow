# ArtPulse 24/7 Operations Runbooks

This directory defines the baseline runbook set for managed ML operations.

- `INCIDENT_RUNBOOK.md`: incident triage, diagnosis, and containment.
- `ROLLBACK_RUNBOOK.md`: model/app rollback procedures (alias + deployment).
- `ONCALL_RUNBOOK.md`: shift model, responsibilities, and handover.
- `ESCALATION_MATRIX.md`: severity-based escalation and response times.

Use these documents as the operating contract for Starter/Growth/Enterprise service packages.
